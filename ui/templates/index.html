What changed in v2.0
New Features
+Exclude Keywords — filter out lines containing specific words from results
+Regex Search — search using regular expressions instead of keywords
+Case Sensitive Mode — toggle exact case matching for keyword searches
+Context Lines — see N lines before and after each match for more context
+Saved Searches — save and reload your frequently used search configurations
+CSV & JSON Download — export results in CSV or JSON format in addition to ZIP and TXT
Fixes & Improvements
~50,000 match cap — search stops at 50k matches to prevent disk full and server crash
~5s no-match timeout — if keyword not found in 5 seconds, search stops with a clear message
~Date filter fixed — results now correctly respect start and end date/time range
~Faster result rendering — results appear immediately after search completes
~Line Range removed — replaced with smarter global match cap (no user input needed)
~ZIP export async — search results show instantly, ZIP builds in background
~Search cancellation — ongoing search can be cancelled cleanly without server crash
